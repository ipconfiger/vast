T22: File upload size limit — Integration Verification
=======================================================

Oversized file via Content-Length header → 413
$ curl -s -o /dev/null -w '%{http_code}' -X POST /api/files/upload \
  -H "Content-Length: 60000000" \
  -H "Content-Type: multipart/form-data; boundary=test" \
  --data-binary "test"
→ 413 (Payload Too Large)

Request is rejected by tower_http::limit::RequestBodyLimitLayer(50*1024*1024)
before reaching the handler.
